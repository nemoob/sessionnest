# CHANGELOG

### v0.2.7(20260721)
#### feature:
1. 新增状态栏完整内容一键截图，生成 PNG 后直接写入剪贴板，支持粘贴到聊天或文档

#### bugFix:
1. 修复截图未完整包含滚动区域、截图失败后残留旧剪贴板内容，以及跟随系统暗色时错误生成亮色截图的问题

### v0.2.5(20260721)
#### feature:
1. 新增“启动时默认打开主窗口”设置，默认关闭，开启后下次启动自动显示主窗口和 Dock 图标

#### bugFix:
1. Token 统计纳入子代理实际用量，并排除子代理日志中复制的父会话历史，避免遗漏或重复计算

### v0.2.4(20260720)
#### feature:
1. 新增版本更新检查，默认每 24 小时最多请求一次 GitHub Release，并支持在菜单栏弹窗查看更新、忽略版本和手动检查；应用不会自动下载或安装更新

#### optimization:
1. 统一菜单栏弹窗刷新按钮与其他顶部按钮的视觉状态，刷新期间保持正常颜色并通过旋转图标提示进度

### v0.2.3(20260720)
#### optimization:
1. 调整为默认仅在 macOS 菜单栏运行，不主动打开主窗口或显示 Dock 图标；从菜单栏弹窗打开主窗口时显示 Dock，关闭主窗口后继续在后台运行

### v0.2.2(20260719)
#### feature:
1. 新增主窗口“额度”一级页面，集中展示 Codex 账户、每周额度、本额度周期本地 Token、额度重置卡次数及逐张到期详情，避免菜单栏入口被系统隐藏时无法查看

#### optimization:
1. 完善额度轻量刷新状态，支持页面打开时按过期策略刷新、手动刷新、最后成功时间和失败提示，并在失败时继续保留上一次成功快照
2. 补充每周额度的本地时区准确重置时间，同时明确服务端额度与本地 Token 统计不能直接换算

### v0.2.1(20260718)
#### bugFix:
1. 修复 Codex linked worktree 按各自绝对路径重复显示同名项目的问题，改为依据 Git common directory 合并到主工作树，并在远端仓库身份唯一时归并已删除 worktree 的历史会话

### v0.2.0(20260718)
#### feature:
1. 新增 Codex 临时会话工作区自动识别，严格匹配 `Codex/YYYY-MM-DD/<会话目录>` 路径，并提供“无项目”侧栏入口集中查看未关联项目的会话
2. 支持从临时会话工作区内的 Git 仓库和会话证据识别真实项目，识别成功时仅展示真实项目，识别不足或候选冲突时归为“无项目”
3. 新增额度重置卡信息展示，可在菜单栏弹窗查看当前可用次数、最近到期时间和逐张到期详情
4. 新增 Token 趋势选点交互，支持悬停预览、单击固定日期，并查看输入、缓存输入、输出、推理输出及总 Token 明细

#### optimization:
1. 重构项目归属缓存模型，明确区分真实项目、普通工作目录和无项目状态，并通过分类器版本触发活跃及归档会话的后台重算
2. 完善项目缓存 SQLite 增量迁移，保留收藏、分类、标签和 Token 缓存，避免要求用户删除本地数据库
3. 调整 Token 统计口径，无项目会话继续计入总量、趋势和会话排行，但不进入项目 Token 排行
4. 修改原“未分类”侧栏文案为“未加入分类”，明确其仍表示尚未加入自定义分类
5. 抽取主窗口与菜单栏共用的 Token 趋势组件，统一点位选择、单点标记、紧凑数字和辅助功能语义
6. 在打开菜单栏弹窗时轻量刷新配额与重置卡信息，并保留原有定时刷新和失败时的上次成功快照
7. 保留并继续写入 v0.1.2 的原每日统计表，以增量新表保存精确 Token 数据，支持旧数据库原位升级和旧版本忽略新表后继续运行

#### bugFix:
1. 修复 Codex 临时会话目录在无法识别 Git 项目时回退为原始 `cwd`，导致大量一次性目录被计入项目树和项目排行的问题
2. 修复同一天使用额度重置卡后“已用 Token”仍包含重置前用量的问题，改为按服务端额度周期的秒级起点统计本地 Token

### v0.1.2(20260717)
#### optimization:
1. 优化菜单栏弹窗的每日 Token 趋势图，在右侧新增纵轴刻度，并使用中文紧凑数字格式展示 Token 数量

### v0.1.1(20260716)
#### optimization:
1. 优化菜单栏弹窗顶部操作区，将官网、GitHub、刷新和打开主窗口统一为带悬停反馈的圆形图标按钮，并增加分隔线改善操作层次
2. 调整官网与 GitHub 的入口图标，使用 Safari 系统图标和 GitHub 标识提升辨识度
3. 调整菜单栏周配额圆环线宽，增强剩余配额在紧凑状态栏中的可读性

### v0.1.0(20260716)
#### feature:
1. 新增原生 macOS Codex 会话管理界面，支持搜索、排序、时间筛选、归档、取消归档和从 Codex 重新打开会话
2. 新增项目目录推断、收藏、自定义分类和彩色标签能力，支持按项目和自定义元数据组织本机会话
3. 新增本地 Token 用量统计，支持按日期、项目和会话查看输入、缓存输入、输出、推理输出及总 Token
4. 新增菜单栏状态与弹窗，支持查看会话数量、Token 覆盖率、周配额剩余比例和当前配额周期 Token 用量
5. 新增 Token 趋势图与项目排行，支持单日点位、多日折线和项目 Token 对比
6. 新增系统、浅色和深色主题切换，支持应用外观跟随 macOS 系统设置
7. 新增本地 SQLite 元数据存储与旧版 `Codex Sessions` 数据库迁移，保留原数据库且不修改 Codex 会话内容

#### note:
1. 明确 SessionNest 采用本机优先架构，不包含遥测、云同步或独立账号系统，收藏、分类、标签和统计结果均保留在当前 Mac

#### script:
1. 新增严格 Swift 格式检查、串行测试、GitHub Actions 持续集成和本地 App 打包签名脚本，支持源码构建与发布前验证
